# AuroraCoder One-Click Launcher

A single binary that deploys AuroraCoder. Download, run, done  — just needs Docker.

## Download

Grab your platform's binary from [releases](https://github.com/thinkwithtool/thinkwithtool/releases).

| Platform    | Download |
|-------------|----------|
| Windows     | `.zip` — extract, then run the `.exe` |
| macOS       | `chmod +x` then run |
| Linux       | `chmod +x` then run |
